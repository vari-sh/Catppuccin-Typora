# 🐱 Catppuccin Mocha for Typora

A soothing pastel theme for [Typora](https://typora.io/), based on the [Catppuccin Mocha](https://github.com/catppuccin/catppuccin) color palette. 

## ✨ Features

- 🎨 **Soothing Colors**: Uses the dark and vibrant Catppuccin Mocha palette to reduce eye strain.
- 🖋️ **Beautiful Typography**: Automatically imports **Inter** for clean prose and **JetBrains Mono** for crisp code blocks via Google Fonts.
- 🗂️ **Enhanced Sidebar**: Features a distinct `mantle` background for the sidebar to separate it from the editor, complete with elegant `mauve` accents when hovering over files.
- 📐 **Full Markdown Support**: Carefully styled tables, blockquotes, code blocks, Mermaid charts, and more.

## 📦 Installation

1. Download the `catppuccin.css` file from this repository.
2. Open Typora.
3. Go to `Settings` > `Appearance`.
4. Click on the **Open Theme Folder** button.
5. Copy/move the `catppuccin.css` file into the opened folder.
6. Restart Typora, or simply select `Catppuccin` from the Themes menu.

##  فون Fonts (Offline Use)

By default, the theme imports [Inter](https://fonts.google.com/specimen/Inter) and [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) from Google Fonts. If you plan to use Typora completely offline, you can remove the `@import` line at the very top of the `catppuccin.css` file and install the fonts locally on your operating system.
